# Omni-Ecom OS 启动手册

## 环境要求

| 依赖 | 最低版本 | 本机安装路径 |
|------|---------|-------------|
| Node.js | 18+ | 已安装 (v24.15.0) |
| JDK | 17+ | `C:\Program Files\Eclipse Adoptium\jdk-17.0.19.10-hotspot` |
| Maven | 3.6+ | `C:\liangtao\ruanjian\apache-maven-3.9.15` |
| MySQL | 8.0+ | `C:\liangtao\ruanjian\phpstudy_pro\Extensions\MySQL8.0.12` |

## 环境变量配置

确保以下环境变量已设置（用户级别，已配置）：

```
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.0.19.10-hotspot
MAVEN_HOME = C:\liangtao\ruanjian\apache-maven-3.9.15
Path 追加 = %JAVA_HOME%\bin;%MAVEN_HOME%\bin
```

> 新开终端后生效。如当前终端未生效，手动刷新：
> ```powershell
> $env:JAVA_HOME = 'C:\Program Files\Eclipse Adoptium\jdk-17.0.19.10-hotspot'
> $env:MAVEN_HOME = 'C:\liangtao\ruanjian\apache-maven-3.9.15'
> $env:Path = "$env:JAVA_HOME\bin;$env:MAVEN_HOME\bin;$env:Path"
> ```

## 启动步骤

### 第一步：启动 MySQL

通过 phpStudy 启动 MySQL 服务，或手动启动：

```powershell
# 确保 MySQL 服务正在运行
& "C:\liangtao\ruanjian\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysqld.exe"
```

验证 MySQL 可连接：

```powershell
& "C:\liangtao\ruanjian\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot -e "SELECT 1"
```

> MySQL 用户名: `root`，密码: `root`

### 第二步：确保数据库存在

```powershell
& "C:\liangtao\ruanjian\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot -e "CREATE DATABASE IF NOT EXISTS digital_employee DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```

如需重新初始化表结构和数据：

```powershell
& "C:\liangtao\ruanjian\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot digital_employee -e "SOURCE C:/liangtao/opencode/omni-ecom-os-v1.0/backend/src/main/resources/data.sql;"
```

### 第三步：启动后端服务 (端口 10004)

```powershell
cd C:\liangtao\opencode\omni-ecom-os-v1.0\backend
mvn spring-boot:run
```

验证后端启动成功：

```powershell
Invoke-RestMethod -Uri 'http://localhost:10004/api/boss' -Method Get
```

> 后端 API 基础地址: `http://localhost:10004/api`

### 第四步：启动前端服务 (端口 10003)

**新开一个终端**，执行：

```powershell
cd C:\liangtao\opencode\omni-ecom-os-v1.0
npm install   # 首次需要安装依赖
npm run dev
```

访问 http://localhost:10003 即可使用系统。

### 后台启动方式（可选）

如需后台运行，不占用终端窗口：

```powershell
# 后台启动后端
Start-Process -FilePath "mvn.cmd" -ArgumentList "spring-boot:run" -WorkingDirectory "C:\liangtao\opencode\omni-ecom-os-v1.0\backend" -WindowStyle Hidden

# 后台启动前端
Start-Process -FilePath "npm.cmd" -ArgumentList "run","dev" -WorkingDirectory "C:\liangtao\opencode\omni-ecom-os-v1.0" -WindowStyle Hidden
```

## 停止服务

```powershell
# 停止后端（占用10004端口的Java进程）
$pid = (Get-NetTCPConnection -LocalPort 10004 -ErrorAction SilentlyContinue).OwningProcess
if ($pid) { Stop-Process -Id $pid -Force }

# 停止前端（占用10003端口的Node进程）
$pid = (Get-NetTCPConnection -LocalPort 10003 -ErrorAction SilentlyContinue).OwningProcess
if ($pid) { Stop-Process -Id $pid -Force }
```

## 数据库配置

配置文件路径: `backend/src/main/resources/application.yml`

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/digital_employee?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: root
```

如 MySQL 密码不同，请修改 `password` 字段。

## 常见问题

### Q: 启动后端报数据库连接失败
确保 MySQL 服务已启动，且 `application.yml` 中的用户名密码正确。

### Q: 前端页面打开但无数据
确保后端服务已启动在 10004 端口。前端 API 地址硬编码为 `http://localhost:10004/api`。

### Q: 端口被占用
```powershell
# 查看10004端口占用
Get-NetTCPConnection -LocalPort 10004
# 查看10003端口占用
Get-NetTCPConnection -LocalPort 10003
```

### Q: Maven 下载依赖慢
可在 `backend/pom.xml` 或 `settings.xml` 中配置阿里云镜像：
```xml
<mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <url>https://maven.aliyun.com/repository/central</url>
</mirror>
```

### Q: 局域网访问
前端 `vite.config.js` 已配置 `host: '0.0.0.0'`，其他设备可通过 `http://你的IP:10003` 访问。后端跨域已通过 `@CrossOrigin(origins = "*")` 放开。
